# 금공러

금융공기업 취준생용 로컬 학습관리 웹앱입니다. 중급회계, 재무관리, 법인세법, 경영학, 경제학, NCS 과목을 기본으로 만들고, 자료 원본 목차와 과목 표준목차를 분리해 관리합니다.

## 핵심 기능

- 브라우저 IndexedDB 저장
- 서버, 로그인, Supabase 없음
- 자료 원본 목차 진도 체크
- 자료 목차와 과목 표준목차 다대다 매핑
- 과목 표준목차로 기록 통합 조회
- 기록함 필터와 카드 복습
- 공부기록 백업, 공부기록 불러오기
- 기본 템플릿 다시 불러오기
- Hash 기반 라우팅과 GitHub Pages 배포 설정
- 기본 PWA manifest/service worker
- 코랄 까마귀 SVG 로고와 파비콘

## 기본 데이터

처음 실행하면 기본 템플릿이 자동 생성됩니다.

- 기본 과목: 중급회계, 재무관리, 법인세법, 경영학, 경제학, NCS
- 기본 자료: IFRS 중급회계 상·하, IFRS 중급회계 강의, 재무회계 기출베스트, 김기동 IFRS 객관식 재무회계, 황윤하 공기업 회계학, 기업은행 경영경제 400제, 세법 자료, 지한송 재무관리 자료
- 중급회계, 재무관리, 법인세법, 경영학, 경제학 표준목차, 대표 세부목차, 대표 매핑 포함

## 로컬 실행

```bash
npm install
npm run dev
```

빌드 확인:

```bash
npm run build
npm run preview
```

## GitHub Pages 배포

`vite.config.ts`에 `base: "./"`가 설정되어 있고, 라우팅은 URL hash를 사용합니다.

1. 이 폴더 내용을 GitHub 저장소 루트로 올립니다.
2. GitHub 저장소 Settings > Pages에서 Source를 `GitHub Actions`로 설정합니다.
3. `main` 브랜치에 push하거나 Actions에서 `Deploy to GitHub Pages` workflow를 수동 실행합니다.
4. 서버 환경변수나 외부 백엔드 설정은 필요 없습니다.

## 백업

설정 화면에서 `공부기록 백업`을 누르면 `.study` 파일로 전체 데이터를 내려받습니다. 새 브라우저나 초기화 이후에는 `공부기록 불러오기`로 다시 가져올 수 있습니다.

## 아이콘

앱 로고와 파비콘은 `public/coral-crow.svg`, `public/favicon.svg`를 사용합니다. 현재 환경에는 PNG 변환 도구가 없어 `icon-192.png`, `icon-512.png`는 만들지 않았습니다. 앱스토어형 설치 아이콘이 필요하면 같은 SVG로 192px, 512px PNG를 생성해 manifest에 추가하면 됩니다.

## 수동 확인 체크리스트

- 앱 실행 후 대시보드가 보이는지
- 첫 실행 시 기본 템플릿이 생성되는지
- 과목 화면에서 중급회계와 재무관리가 보이는지
- 자료 화면에서 기본 6개 자료가 보이는지
- 자료 목차에서 시작전 / 진행중 / 완료 변경이 되는지
- 자료 목차에서 기록 추가가 되는지
- 과목 목차에서 여러 자료의 기록이 통합 조회되는지
- 기록함에서 과목, 표준목차, 세부목차, 자료, 자료 목차, 기록유형, 학습상태, 작성일 필터가 작동하는지
- 카드 복습에서 버튼과 방향키로 앎 / 모름 처리가 되는지
- 모름 처리한 기록이 대시보드의 모르는 기록 위젯에 보이는지
- 공부기록 백업과 공부기록 불러오기가 되는지
- 모바일 폭에서 하단 탭과 기록함, 카드 복습 화면이 깨지지 않는지

## 1차 MVP 범위

서버형 공유, 로그인, 실시간 동기화는 제외했습니다. `공유하기` 버튼은 추후 읽기 전용 링크 방식으로 확장할 위치를 잡아 둔 안내 모달입니다.
